# Unity 休闲益智游戏 - 整理专家
**开发专业**：数字媒体技术
**开发引擎**：Unity 2018.3.6f1
**运行平台**：PC
**核心玩法**：基于物品匹配消除的拖拽式整理，结合关卡挑战、策略规划与趣味剧情

## 项目介绍
本游戏为自主设计开发的休闲益智类作品，参考微信小程序《消消奶茶屋》核心机制，完成**全维度原创重构与功能升级**，打造专属「熊猫强迫症整理店」主题。
游戏以熊猫「团团」为主角，设计5级差异化关卡体系，从基础教学到无尽挑战，逐步提升游戏难度与策略性，兼具操作趣味性与玩法深度。

## 核心特色
### 1. 原创主题与剧情
- 打造「熊猫强迫症整理店」独家主题，设计熊猫主角「团团」及专属对话剧情
- 内置完整教学引导系统，通过角色对话自然传递游戏规则，提升沉浸感

### 2. 差异化关卡体系（5级）
| 关卡类型 | 核心挑战 | 专属机制 |
|----------|----------|----------|
| 教学关   | 基础操作 | 3类基础物品，无补货 |
| 基础关   | 节奏把控 | 5类物品，8秒自动补货+时间限制 |
| 策略关   | 空间规划 | 6类物品，中央格子损坏+10秒补货 |
| 速度关   | 快速反应 | 8类物品，连击加分系统+10秒补货 |
| 无尽关   | 持久整理 | 10类物品，无时间限制+9秒补货 |

### 3. 核心功能优化（原创开发）
- **智能补货系统**：安全类型算法+批量生成策略，避免游戏不平衡与卡死
- **精准拖拽系统**：鼠标跟随优化+智能放置算法，贴合玩家操作意图
- **数据持久化**：支持关卡最高分、星星评级、解锁状态存档，进度自动同步
- **动态UI系统**：熊猫对话打字机效果、星星动态显示、按钮音效自动绑定
- **性能优化**：延迟执行+内存管理+协程控制，保证游戏流畅运行

### 4. 美术与特效原创改进
- 为10类食物设计**风格统一的3D图标**，优化碰撞器与中心点，拖拽带流畅动画
- 定制专属特效：消除缩放淡出、星星评级颜色变化、损坏格子红色永久标记
- UI界面视觉统一：关卡选择、暂停/结束菜单、对话系统风格一致，功能完整

## 游戏运行说明
1. 下载本仓库所有文件，使用**对应版本Unity**打开项目根目录；
2. 在Unity中找到**主场景**（如Scenes/Start），点击运行即可开始游戏；
3. 操作方式：鼠标拖拽物品至货架空位，三个相同物品在同一个格子中完成消除，累计分数通关。

## 核心玩法规则
1. 基础消除：三个相同物品匹配消除，单次消除得100分；
2. 连击加分：3秒内连续消除触发连击，额外获得50分/次；
3. 通关条件：在时间限制内达到关卡目标分数，获得星星评级（至少1颗星解锁下一关）；
4. 自动补货：关卡开启后按固定间隔自动补货，补货规则随关卡调整。
